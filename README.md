# Omics_analysis
URLs that could be interesting for Omics Analysis.

* [URLs of Metabolomics and omics tools](https://github.com/RASpicer/MetabolomicsTools)
* [KEGG: DATABASE OF GENES and their pathways](https://www.kegg.jp/kegg-bin/search_pathway_text?map=map&keyword=KRAS&mode=1&viewImage=true)
* [GeneCards: Database of Genes and their description](https://www.genecards.org/Search/Keyword?queryString=KRAS)
* [Variant Effect Predictor](http://grch37.ensembl.org/info/docs/tools/vep/script/vep_example.html)
* [Gene text: Describes gene expression](https://gtexportal.org/home/)
* [String-db: Pathway visualisation of Genes and proteins](https://string-db.org/)
* [UCSC: Genome browser](https://genome.ucsc.edu/)
* [Integrative Genome Viewer](https://igv.org/app/)
